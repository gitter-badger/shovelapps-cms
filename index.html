--- layout: default --- #Getting Started ## Prerequisites Before you begin, make sure you have [Node.js](http://nodejs.org/download) installed. Download the latest version. ``` $ wget https://github.com/shovelappsinc/shovelapps-cms/archive/latest.zip $ cd ~/shovelapps-cms ``` * Unzip it and run it. ``` $ unzip shovelapps-cms-latest.zip $ cd shovelapps-cms-latest/ $ ./start.sh ```
